Weekly Data Pipeline

The dashboard reads from the SNAPS array at the top of the script. To update weekly, append a new snapshot object every Monday. Here's the structure:

{
  date: "2026-04-21",          // ISO date string
  label: "Apr 21, 2026",       // Human-readable
  igFollowers: [13000,...],    // 8 values, one per competitor
  pricePerOz:  [1.88,...],     // Current price/oz
  reviewScore: [4.4,...],      // Avg review score
  reviewCount: [120,...],      // Total review count
  inStockPct:  [100,...],      // In-stock % (0–100)
  newPress:    [1,...],        // Press mentions this week
  retailDoors: [200,...],      // Est. retail doors
  sentiment: {
    amazon:   {score:4.4, count:120, mentions:14, trend:"up"},
